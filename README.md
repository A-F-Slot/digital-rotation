# Digital Rotation - GTUD Scientific Data

This repository contains the scientific data for the GTUD (Generic Test and User Data) Digital rotation experiments, specifically versions V6-2 and V6-3.

## Overview

This repository accompanies a research paper and provides the datasets, metadata, and documentation necessary for reproducibility and reuse of the experimental results.

## Repository Structure

```
digital-rotation/
├── data/                      # Data files
│   ├── raw/                   # Original, unprocessed data
│   ├── processed/             # Cleaned and processed data
│   └── metadata/              # Metadata describing the datasets
├── LICENSE                    # License information (GPL-3.0)
├── CITATION.cff              # Citation information
└── README.md                 # This file
```

## Data Description

The repository contains data from two experimental versions:
- **V6-2**: Digital rotation experiment version 6-2
- **V6-3**: Digital rotation experiment version 6-3

For detailed information about the data files, formats, and variables, please see the [data/README.md](data/README.md) file.

## Usage

### Accessing the Data

1. Clone this repository:
   ```bash
   git clone https://github.com/A-F-Slot/digital-rotation.git
   cd digital-rotation
   ```

2. Navigate to the `data/` directory to access the datasets:
   - `data/raw/` contains original data files
   - `data/processed/` contains processed data ready for analysis
   - `data/metadata/` contains metadata files describing the datasets
