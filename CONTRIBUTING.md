# Contributing to Digital Rotation Data Repository

Thank you for your interest in contributing to this scientific data repository!

## How to Contribute

### Reporting Issues

If you find any issues with the data, metadata, or documentation:

1. Check if the issue has already been reported in the [Issues](https://github.com/A-F-Slot/digital-rotation/issues) section
2. If not, create a new issue with:
   - A clear, descriptive title
   - A detailed description of the issue
   - Steps to reproduce (if applicable)
   - Expected vs. actual behavior

### Suggesting Enhancements

We welcome suggestions for:
- Additional metadata
- Improved documentation
- Data format improvements
- Better organization

Please open an issue with the label "enhancement" to discuss your ideas.

### Contributing Data or Corrections

If you have corrections to the data or additional related data:

1. Fork the repository
2. Create a new branch (`git checkout -b data-correction`)
3. Make your changes
4. Include detailed documentation of what changed and why
5. Submit a pull request

Please ensure:
- All changes are well-documented
- Metadata is updated accordingly
- The README is updated if necessary

### Data Quality Standards

All data contributions should:
- Be clearly documented
- Include appropriate metadata
- Use open, non-proprietary formats when possible
- Respect data privacy and ethical guidelines
- Be properly licensed

## Questions?

If you have questions about contributing, please open an issue or contact the repository maintainer.

## Code of Conduct

Please be respectful and constructive in all interactions. This is a scientific repository meant to facilitate open research and collaboration.
